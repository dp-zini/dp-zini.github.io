hitman internet blocking, automated!

november 23, 2023 @ 3:52

it blocks hitman from accessing the internet so you can exit freelancer/elusive target without failing.

steps:

1. create firewall rules:
- open the search bar and type "windows defender firewall".
- select "outbound rules", then "new rule".
- select "program" and "next".
- go to your hitman 3 installation folder, "retail" subfolder, and select "hitman3.exe".
- choose "block the connection", click "next", and name it "hitman".

2. download and run the python script:
- download the script.
- after downloading, run the script. it will create the .bat files.

3. using the batch files:
- `blockhitman.bat` blocks internet access.
- `unblockhitman.bat` restores internet access.

4. shortcut keys with autohotkey:
- it also makes the ahk's.
- press alt + f2 to block.
- press alt + f1 to unblock.

note: make sure you have autohotkey installed for using the shortcut keys.
